# RF-F-OPTICAL-ENERGY-V1 source provenance

All numeric profiles are sensitivity envelopes. Values are not process- or line-rate-normalized, because doing so without a validated scaling law would create false precision. The profile preserves each source's node/process, units, rate, wavelength scope, and exact conversion.

Receiver margins (1/3/6 dB) are explicit model guardbands, not measured values from the cited receiver papers. The nominal/high thermal choices assume one/two actively tuned resonators per provisioned TX lane; the low profile explicitly selects a passive/athermal realization.
